# Application Support Incident Management System

This project simulates a real-world Application Support incident tracking system used by IT support teams.

## Objective

To capture, track, and manage application incidents reported by users through a centralized ticketing system.

## Tools Used

- Google Forms
- Google Sheets
- Excel Dashboard
- Google Workspace

## Features

- Incident submission form
- Automated incident ID generation
- Centralized ticket database
- Priority-based issue tracking
- Ticket status management (New, In Progress, Closed)
- Incident reporting dashboard

## Workflow

1. Users submit issues through Google Form
2. Responses are logged in Google Sheets
3. Incident ID is automatically generated
4. Support analysts assign tickets and update status
5. Dashboard provides analytics on incidents
